## ✅ 使用提示

* **不同地区做了访问优化**：某些入口可能在你的网络环境下更快；打不开就换下一个入口。
* **浏览器建议**：

  * iOS/macOS：优先使用 **Safari** 打开
  * Android：优先使用 **Edge 或 Chrome** 打开
  * Windows：使用 **Edge/Chrome** 最新版
* **收藏防失联**：建议在浏览器中按 `Ctrl+D`（Windows）或 `⌘+D`（macOS）将入口页加入书签。
* **不要在微信或 QQ 内直接打开**，可能拦截外链或导致下载失败。
